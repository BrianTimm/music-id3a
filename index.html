music-id3a by BrianTimm

A music sorting script written for the Bash shell

music-id3a is the result of my own inability to deal with an unorganized music collection. I have >10k songs and not having some kind of order was beginning to really annoy. so I started on music-id3a.

Dependencies

music-id3a requires: - a UNIX/LINUX Operating System - a Bash shell (similar may work, I haven't tested it though) - id3v2 which you can get prepackaged (at least on Ubuntu you can)

Install

Modify the music-id3a.cfg file to suit your needs/settings. ensure that id3v2 is installed and that you can write your music folder. Other than that, check the README for more thorough instructions.
